## How to start app
* run `sudo npm install --global yarn` to get yarn
* then run `yarn install`
* then run `yarn start`
* go to http://localhost:8000/


###
Table/Chart/Graph Frameworks:
* React-Chartjs-2
* Chart.js

### to list dependencies
`yarn list`
`yarn list --depth=0`
`yarn list --depth=0 --pattern <dependency name>`
